# 📚 Sistema de Registro de Atrasos Escolares

Sistema desktop desenvolvido em **Python + Tkinter** para registro, gerenciamento e impressão de atrasos de estudantes.

Ideal para uso em instituições de ensino que utilizam impressora térmica 80mm (ex: GP-C80250I).

---

## 🚀 Funcionalidades

### 👨‍🎓 Cadastro Inteligente de Alunos
- Salvamento automático em `alunos.json`
- Autocomplete enquanto digita matrícula ou nome
- Atualização de matrícula e nome
- Sugestões automáticas de alunos existentes

---

### ⏰ Registro de Atrasos
- Data automática
- Hora do registro automática
- Horário de início da aula
- Horário de chegada
- Docente responsável
- Turma
- Motivo do atraso
- Salvamento em `atrasos.json`

---

### 🖨 Impressão Térmica 80mm
Compatível com impressoras térmicas padrão ESC/POS.

- Layout centralizado e organizado
- Margem ajustada para corte correto
- Corte automático no final
- Codificação ajustada para evitar erro de acentuação

---

### 📊 Aba de Registros Avançada

- Listagem completa dos atrasos
- Agrupamento automático por aluno
- Campo de busca por matrícula ou nome
- Botão editar registro
- Botão excluir registro
- Botão cancelar
- Exibição de:
  - Data
  - Hora do registro
  - Matrícula
  - Nome
  - Docente
  - Turma
  - Início da aula
  - Chegada
  - Motivo

# ENGLISH
# 📚 Student Late Arrival Registration System

Desktop application developed using **Python + Tkinter** for registering, managing, and printing student late arrivals.

Ideal for educational institutions using 80mm thermal printers (e.g., GP-C80250I).

---

## 🚀 Features

### 👨‍🎓 Smart Student Management
- Automatic saving to `students.json`
- Autocomplete while typing student ID or name
- Update student ID and name
- Automatic suggestions for existing students

---

### ⏰ Late Arrival Registration
- Automatic date
- Automatic registration time
- Class start time
- Student arrival time
- Responsible teacher
- Class/group
- Reason for lateness
- Saved in `late_records.json`

---

### 🖨 80mm Thermal Printing
Compatible with ESC/POS standard thermal printers.

- Centered and organized layout
- Adjusted margins for proper cutting alignment
- Automatic paper cut
- Encoding adjusted to prevent accent character issues

---

### 📊 Advanced Records Tab

- Full list of late arrival records
- Automatic grouping by student
- Search field by student ID or name
- Edit record button
- Delete record button
- Cancel button
- Displays:
  - Date
  - Registration time
  - Student ID
  - Student name
  - Teacher
  - Class/group
  - Class start time
  - Arrival time
  - Reason
