# 📚 Sistema de Registro de Atrasos Escolares

Sistema desktop desenvolvido em **Python + Tkinter** para registro, gerenciamento e impressão de atrasos de estudantes.

Ideal para uso em instituições de ensino que utilizam impressora térmica 80mm (ex: GP-C80250I).

---

## 🚀 Funcionalidades

### 👨‍🎓 Cadastro Inteligente de Alunos
- Salvamento automático em `alunos.json`
- Autocomplete enquanto digita matrícula ou nome
- Atualização de matrícula e nome
- Sugestões automáticas de alunos existentes

---

### ⏰ Registro de Atrasos
- Data automática
- Hora do registro automática
- Horário de início da aula
- Horário de chegada
- Docente responsável
- Turma
- Motivo do atraso
- Salvamento em `atrasos.json`

---

### 🖨 Impressão Térmica 80mm
Compatível com impressoras térmicas padrão ESC/POS.

- Layout centralizado e organizado
- Margem ajustada para corte correto
- Corte automático no final
- Codificação ajustada para evitar erro de acentuação

---

### 📊 Aba de Registros Avançada

- Listagem completa dos atrasos
- Agrupamento automático por aluno
- Campo de busca por matrícula ou nome
- Botão editar registro
- Botão excluir registro
- Botão cancelar
- Exibição de:
  - Data
  - Hora do registro
  - Matrícula
  - Nome
  - Docente
  - Turma
  - Início da aula
  - Chegada
  - Motivo
